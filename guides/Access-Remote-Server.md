# Access Remote Server

<img src="https://wgdashboard-resources.tor1.cdn.digitaloceanspaces.com/Documentation%20Images/user-guides/access-remote-server.webp" style="block" alt=""/>

## Enable

To access remote server, simply toggle the **switch** under **Sign In** button in the [](Sign-In.md) page

<video autoplay loop controls src="https://wgdashboard-resources.tor1.cdn.digitaloceanspaces.com/Documentation%20Images/user-guides/enable-access-remote-server.mp4"></video>

## Add Remote Server

To add remote server, simply toggle the switch, click **+ Server** button, fill in the address and API Key.

<video autoplay loop controls src="https://wgdashboard-resources.tor1.cdn.digitaloceanspaces.com/Documentation%20Images/user-guides/add-remote-server.mp4"/>

### Enter Remote Server

Once you have address and API key filled in, you will see an indicator beneath the inputs telling you if your remote server's status.

#### If success

![remote-server-success.webp](https://wgdashboard-resources.tor1.cdn.digitaloceanspaces.com/Documentation%20Images/user-guides/remote-server-success.webp)

#### If failed

Usually caused by your address can't be connected, or API Key issue.

![remote-server-failed.webp](https://wgdashboard-resources.tor1.cdn.digitaloceanspaces.com/Documentation%20Images/user-guides/remote-server-failed.webp)
